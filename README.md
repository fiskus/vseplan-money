Vseplan.Money: incomes/expenses tracker

![demo](public/assets/demo.gif)

## Development

`npm install`
`cp .env.example .env`
`npm run dev`


## Features

* Track incomes, wallets and expenses
* Mobile version
* Values adjustments
* Coinkeeper export
* Facebook auth

## Short term roadmap:

* Deployment
* Move money from one wallet to another
* Currencies
* Export from other popular money trackers
* Google and Github auth
